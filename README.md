# 🦄 Git Wrapped

> 程序员的赛博年度总结 —— 把 Git 日志变成可分享的摸鱼报告。

[![Go Version](https://img.shields.io/badge/Go-1.20+-00ADD8?style=flat&logo=go)](https://go.dev/)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

## ✨ 特色

- 🔒 **100% 离线运行** —— 你的代码数据绝不离开电脑
- 📊 **摸鱼指数分析** —— 统计 `fix typo`、`.` 等划水 Commit
- 🌙 **夜猫子指数** —— 凌晨写代码的占比
- 📅 **7x52 热力图** —— ASCII 艺术风格，展示全年活跃度
- 💕 **文件 CP 组合** —— 最常一起修改的文件对
- 🔥 **卷王指数** —— 最长连续提交天数
- 🛠️ **重构狂魔** —— 被频繁修改的文件统计
- 🎨 **终端彩虹 UI** —— 基于 Lipgloss
- 📸 **HTML 社交名片** —— 自动生成并打开浏览器，适合截图分享
- 📦 **多仓库聚合分析** —— 一键分析整个 coding 目录
